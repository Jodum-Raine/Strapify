# Strapify Ordering System

## Overview
A custom ordering system built specifically for **Strapify**, a strap and accessories business. This system replaces manual order-taking with a digital solution that streamlines the customer ordering process and simplifies order management for the business owner.

## Features

### Customer Features
- Browse products with categories
- Add items to shopping cart
- Place orders with customer details
- Track order status

### Admin Features
- Admin dashboard to view all orders
- Update order status


## Technologies Used
- Frontend: HTML, CSS, Tailwind CSS, JavaScript
- Backend: PHP
- Database: MySQL

## How to Run Locally
1. Clone this repository
2. Import the database from `database/strapify_db.sql`
3. Update database credentials in `includes/config.php`
4. Run on localhost
